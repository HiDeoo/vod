# Requirements

- [Streamlink](https://streamlink.github.io/)

# Installation

At the moment, the application is not yet packaged into an executable.

1. Clone the repository.
1. `yarn`
1. `yarn build`
1. Create a copy of the `.env` file named `.env.local` and fill it.

## Usage

```sh
$ yarn start <channel>
```
